# Battle Pass

## Overview

The Battle Pass is a 100-tier seasonal progression track lasting 90 days. It contains a free track (30 items) and a premium track ($9.99, 70 additional items). The Battle Pass is the primary seasonal content delivery mechanism for cosmetics.

## Season Structure

| Parameter | Value |
|-----------|-------|
| Season duration | 90 days |
| Total tiers | 100 |
| Free track items | 30 (at tiers 1, 4, 7, 10, ... every 3rd tier) |
| Premium track items | 70 (remaining tiers) |
| Premium price | $9.99 USD |
| Level buying | Not available (must earn XP) |
| Retroactive unlock | Yes (buying premium mid-season grants all earned premium tiers) |

## XP Sources

| Source | XP Earned | Frequency |
|--------|-----------|-----------|
| Delve completion (full clear) | 500 XP | Per run |
| Delve completion (partial) | 200 XP | Per run |
| Zone boss kill | 100 XP | Per boss per run |
| Daily challenge | 300 XP | 1 per day |
| Weekly challenge | 1,500 XP | 3 per week |
| Heat clear bonus | +50 XP per Heat level | Per run |
| First run of the day | 200 XP bonus | Once daily |
| Codex entry unlock | 150 XP | Per entry |

### XP Per Tier

| Tier Range | XP per Tier | Reasoning |
|-----------|-------------|-----------|
| 1-25 | 1,000 XP | Fast early progression, engagement hook |
| 26-50 | 1,500 XP | Moderate pacing |
| 51-75 | 2,000 XP | Slower, players committed |
| 76-100 | 2,500 XP | Endgame grind, reward dedicated players |

### Total XP Required

| Milestone | Cumulative XP | Est. Play Hours |
|-----------|--------------|-----------------|
| Tier 25 | 25,000 | ~20 hours |
| Tier 50 | 62,500 | ~50 hours |
| Tier 75 | 112,500 | ~85 hours |
| Tier 100 | 175,000 | ~120 hours |

A player completing daily + weekly challenges and 1-2 runs/day should reach Tier 100 in approximately 80-90 days (before season end).

## Post-Season Item Availability

| Item Tier | Post-Season Availability |
|-----------|------------------------|
| Free track items | Available in store immediately |
| Premium recolors/emotes | Available in store after 1 full season (90 days) |
| Themed skins | Available in store after 1 full season |
| Animated skins | Available in store after 1 full season |

| Titles | Permanently exclusive to original season |

## Anti-FOMO Design

| Principle | Implementation |
|-----------|---------------|
| No level buying | Players cannot purchase tiers; engagement required |
| Retroactive premium | Buying premium at any point unlocks earned tiers |
| Items return to store | Nothing permanently exclusive except titles |
| Generous XP curve | Casual players (1 hr/day) can reach Tier 75+ |
| No gameplay advantage | All items are cosmetic only |
