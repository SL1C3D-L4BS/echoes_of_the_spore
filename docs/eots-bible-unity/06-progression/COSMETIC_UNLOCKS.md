# Cosmetic Unlocks

## Design Principles

All cosmetics are visual-only. No cosmetic provides gameplay advantage. Cosmetics communicate player achievement and identity. Every cosmetic has a clear unlock source so players know what to pursue.

|----------|-------------------|-------------------|
| Character Skins | 48 (12 per class) | 1 equipped |

## Fusion Skins

| Fusion Pair | Default | Unlockable |
|-------------|---------|-----------|
| Sporeling + Rootwarden | Myco-Juggernaut (base) | Ironspore variant (Codex: 100 Fusion kills) |
| Sporeling + Bloomcaster | Myco-Tempest (base) | Stormbloom variant (Codex: 50 Spore Plans) |
| Sporeling + Decayer | Myco-Wraith (base) | Shadowspore variant (Heat 20 fused clear) |
| Rootwarden + Bloomcaster | Myco-Bastion (base) | Petalguard variant (Codex: 10hr Fusion time) |
| Rootwarden + Decayer | Myco-Revenant (base) | Rotshield variant (Heat 30 fused clear) |
