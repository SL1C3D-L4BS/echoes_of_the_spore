# Echo Codex

## Overview
300+ entries tracking player discovery and mastery. Five categories, each with unique unlock conditions and rewards.

## Categories

### 1. Symbiosis (60 entries)
Tracks Fusion and cooperative play.

| Entry Type | Count | Unlock Condition | Example |
|-----------|-------|-----------------|---------|
| Fused Class discovery | 12 | First time fusing each combination | "Myco-Juggernaut: Fuse Sporeling + Rootwarden" |
| Fused combat kills | 12 | Kill 100 enemies as each Fused class | "Juggernaut Centurion: 100 kills as Myco-Juggernaut" |
| Spore Plan executions | 8 | Execute 50 Spore Plans per scenario | "Coordinated Strike: 50 offensive Spore Plans" |
| Fusion duration milestones | 8 | Stay fused for cumulative time | "Inseparable: 10 hours total Fused time" |
| Partner achievements | 12 | Achieve specific combos with partners | "Symbiotic Resonance: Clear Zone 8 without separating" |
| Rite of Echoes | 8 | PvP milestones | "Rite Champion: Win 10 Rites" |

### 2. Corruption (50 entries)
Tracks Infection and voice corruption.

| Entry Type | Count | Unlock Condition |
|-----------|-------|-----------------|
| Infection tier milestones | 10 | Reach each 10% tier in a single run |
| Voice corruption discoveries | 8 | Trigger specific DSP effects |
| Purification records | 8 | Purify at various costs |
| Infection + combat combos | 12 | Kill bosses at high Infection |
| Infection endurance | 12 | Complete zones without purifying |

### 3. Civilization (80 entries)
Tracks lore discovery and boon mastery.

| Entry Type | Count | Unlock Condition |
|-----------|-------|-----------------|
| Fossil Mural discoveries | 40 | Find all 40 murals |
| Civilization boon mastery | 8 | Use 10+ boons from each civilization |
| Duo Echo discoveries | 28 | Trigger all 28 Duo combinations |
| Legendary Echo discoveries | 8 | Trigger all 8 Legendaries |

### 4. Community (50 entries)
Tracks Community Tree and server participation.

| Entry Type | Count | Unlock Condition |
|-----------|-------|-----------------|
| Tree contribution milestones | 20 | Contribute cumulative Echoes |
| Branch depth milestones | 20 | Be online when specific depths unlock |

| True Ending | 5 | Vote in True Ending, witness outcome |

### 5. Mastery (70 entries)
Tracks Heat system and mechanical skill.

| Entry Type | Count | Unlock Condition |
|-----------|-------|-----------------|
| Heat milestones per class | 20 | Clear at Heat 10/20/30/50 per class |
| Speed records | 16 | Clear zones under par time per Heat |
| No-hit achievements | 8 | Clear boss without taking damage |
| Condition-specific clears | 15 | Clear with specific Condition combos |
| Ultimate challenges | 11 | Combined mastery (e.g., "Heat 50, no boons, solo") |

## Reward Distribution
- Every 10 entries: cosmetic unlock (skin, VFX, title)
- Every 50 entries: major cosmetic (animated skin, Fusion effect)
- 100%: "Echo Archivist" title + unique Canopy monument
- Entries are permanent (never reset, never expire)
