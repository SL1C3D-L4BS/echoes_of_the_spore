# COMPETITIVE ANALYSIS

## Tier 1 Competitors (Direct -- Same Genre Space)

### Hades II (Supergiant Games)
| Attribute | Detail |
|---|---|
| Launch | September 2025 (1.0) |
| Price | $29.99 |
| Sales | Multi-million units |
| Engine | Custom (Supergiant proprietary) |
| Strengths | Best-in-class combat feel, exceptional narrative integration, Supergiant brand loyalty, stunning art |
| Weaknesses | Singleplayer only, no co-op, fixed world (no persistence), replay value plateaus after story completion |
| What Echoes Learns | Combat feel targets (hitstop, input responsiveness). Boon system depth. Narrator as reactive storytelling. |
| How Echoes Differentiates | Multiplayer fusion, persistent world, infection comedy, Community Tree |

### Dead Cells (Motion Twin / Evil Empire)
| Attribute | Detail |
|---|---|
| Sales | 15M+ lifetime |
| Price | $24.99 (base) |
| Strengths | Tight combat, excellent animation, strong DLC pipeline, responsive controls |
| Weaknesses | Singleplayer only, no meaningful persistence between runs beyond unlocks, aging (2018 launch) |
| What Echoes Learns | Movement speed tuning, weapon variety, biome visual identity |
| How Echoes Differentiates | Co-op, persistent shared world, unique fusion mechanic |

### Hollow Knight: Silksong (Team Cherry)
| Attribute | Detail |
|---|---|
| Launch | September 2025 |
| Sales | 7M+ lifetime |
| Price | $14.99 |
| Strengths | World-class level design, tight combat, extraordinary value per dollar, massive audience |
| Weaknesses | Singleplayer only, no procedural generation, no co-op, no persistence |
| What Echoes Learns | Mask HP system, Silk/Soul healing, environmental storytelling, boss design philosophy |
| How Echoes Differentiates | Multiplayer, roguelite replayability, living world |

### Slay the Spire 2 (Mega Crit Games)
| Attribute | Detail |
|---|---|
| Launch | March 2026 (Early Access) |
| Sales | 3M week 1 |
| Price | $24.99 |
| Engine | Godot |
| Strengths | Genre-defining deckbuilder, 4-player co-op (new), massive community, near-perfect reviews |
| Weaknesses | Turn-based (no action), card-based (different skill ceiling), no Metroidvania exploration |

## Lessons from Failures

| Title | What Went Wrong | Lesson for Echoes |
|---|---|---|
| Godfall | Combat was beautiful but shallow; no co-op loop | Depth must come from systems interaction, not just visuals |
| Babylon's Fall | Live-service fatigue, $60 + MTX | Never position as live-service; premium-first |
| Bleeding Edge | Multiplayer focus with no solo path | Always have viable solo experience |
| Exoprimal | PvPvE confused identity | Clear genre communication in marketing |
| Hyper Light Breaker | Open-world roguelite lost the tight loop | Maintain tight 10-40 min run structure |

## Release Window Strategy

| Window | Risk | Opportunity |
|---|---|---|
| Q1 2028 (current target) | Post-holiday slump, lower organic traffic | Less competition, longer runway for marketing |
| Q2 2028 | Summer Game Fest visibility | Competing with E3/SGF announcements |
| Q3 2028 | Pre-holiday positioning | Risk of being buried by AAA fall releases |
| Q4 2028 | Holiday sales momentum | Extremely competitive window |

**Recommendation:** Q1 2028 launch. Avoid Q4 entirely. Q1 has historically been strong for indie breakouts (Hades 1.0 launched September, but many roguelites thrive in Q1 when AAA pipeline is thin).
