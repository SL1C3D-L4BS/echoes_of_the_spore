# Seasonal Content Pipeline

## Season Structure (90 Days Each)

| Week | Content Drop | Type |
|------|-------------|------|
| 1 | Season launch: Battle Pass, new boons, balance patch | Major |
| 3 | Community challenge #1 | Event |
| 5 | Mid-season patch: bug fixes, tuning | Patch |
| 7 | Community challenge #2 | Event |
| 9 | Rite of Echoes tournament | Event |
| 11 | Late-season patch: QoL, community requests | Patch |
| 13 | Season end: rewards distributed, next season teaser | Major |

## Battle Pass Tiers (100 Levels)

### Free Track (30 Items)
| Tier Range | Rewards |
|-----------|---------|
| 1-10 | 3 basic recolors, 500 Echoes, 2 titles |
| 11-30 | 3 advanced recolors, 1000 Echoes, 2 emotes, 1 Canopy decoration |
| 31-50 | 2 unique recolors, 1500 Echoes, 1 animated border |
| 51-70 | 2 rare recolors, 2000 Echoes, 1 class icon variant |
| 71-100 | 1 exclusive skin, 3000 Echoes, 1 title, season badge |

### Premium Track ($9.99 — 70 Additional Items)
| Tier Range | Rewards |
|-----------|---------|
| 1-25 | Animated skins (1 per class), Fusion VFX, 5000 Echoes |
| 26-50 | Infection overlay variants, Spore-Talk voice filter, emotes |
| 51-75 | Exclusive weapon effects, death animations, loading screens |
| 76-100 | Legendary skin, animated profile banner, season finale cosmetic |

### XP Sources

| Action | XP | Daily Cap |
|--------|-----|-----------|
| Complete a delve | 100 | No cap |
| Clear a boss | 50 | No cap |
| Contribute to Community Tree | 25 per 1000 Echoes | 200 XP |
| Win Rite of Echoes | 150 | 300 XP |
| Daily challenge (3/day) | 50 each | 150 XP |
| Weekly challenge (3/week) | 200 each | 600 XP |

### XP Per Level
```
Level 1-20:   1,000 XP each
Level 21-50:  1,500 XP each
Level 51-80:  2,000 XP each
Level 81-100: 2,500 XP each
Total: 165,000 XP to complete
```

Average player earning ~500 XP/day completes in ~90 days (full season). Dedicated players finish in ~45 days.
