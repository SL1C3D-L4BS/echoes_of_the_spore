## UI Color Usage Rules

1. **Void Black** (#0A0A0F) is the universal background — never pure #000000
2. **Spore White** (#E8E4D9) is the text color — never pure #FFFFFF
3. **Infection Cyan** is reserved for health/infection only — never decorative
4. **Necrospore Gold** is the reward color — only for loot, XP, milestones
5. **Damage numbers** use attack type color (white normal, cyan crit, gold Duo Echo)
6. **Status effects** always use their civilization's color
7. **Never use more than 3 palette colors in one UI panel**
