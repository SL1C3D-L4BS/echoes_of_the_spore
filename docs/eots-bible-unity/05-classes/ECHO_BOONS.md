# ECHO BOONS -- Complete Reference

## System Overview

8 civilizations, 5 boon slots, 4 rarities. Boons are offered at Reward Rooms during a run. Each boon occupies one of 5 slot types. Slots are exclusive per civilization (only one boon per slot per civilization at a time). Picking a new boon from the same civilization and slot replaces the old one.

## Slot Types

| Slot | Function | Typical Effect |
|---|---|---|
| Strike | Modifies basic attacks | Damage bonuses, hit effects, elemental additions |
| Surge | Modifies dash behavior | Dash damage, dash distance, invulnerability frames |
| Spore Cast | Modifies Active Function abilities | CD reduction, damage increase, AoE expansion |
| Symbiote | Defensive / survivability | HP regen, shields, damage reduction |
| Resonance | Meta / stacking bonus | Scales with other boons, combo incentives |
