# ECHO FUNCTIONS -- Complete Reference

## Overview

32 total Echo Functions (8 per class). Each Function operates in 3 modes: Active, Upgrade, and Passive. Using a single Function in all 3 modes across separate runs unlocks its associated lore entry (96 lore entries total).

Functions are acquired from Echo Rooms during a run. Players slot them according to their Crest's slot layout.

## Mode Definitions

| Mode | Slot Type | Behavior |
|---|---|---|
| Active | A slot | Activatable ability with its own cooldown |
| Upgrade | U slot | Attaches to an Active, adding a secondary effect triggered alongside it |
| Passive | P slot | Permanent stat bonus or triggered effect, always on |

## Lore Unlock Tracker

Each Function has a lore entry unlocked by using it in all 3 modes across any number of runs:

| Completion | Reward |
|---|---|
| 1 Function (all 3 modes) | That Function's lore entry |
| 8 Functions (1 full class) | Class Origin story chapter |
| 16 Functions (2 classes) | Civilization cross-reference entry |
| 32 Functions (all classes) | "Voice of the Mycelium" achievement + secret Echo Boon |

## Function Acquisition

- Functions drop from Echo Rooms (guaranteed 1 per room, choice of 3).
- Rarity does not apply to Functions themselves; rarity applies to Echo Boons.
- A Function can only be equipped once per loadout (no duplicates).
- Unequipping a Function mid-run is allowed at Rest Sites.
