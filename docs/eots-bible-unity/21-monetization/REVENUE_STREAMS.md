## Monetization Ethics

| Rule | Implementation |
|------|---------------|
| No pay-to-win | All purchases cosmetic-only |
| No loot boxes | Every purchase is a known item at a known price |
| No pay-to-skip | Cannot buy Heat levels, Echo Functions, Codex entries |
| No FOMO pressure | Battle Pass items return to store after 1 season |
| No dark patterns | No "limited time" countdowns, no manipulative pricing |
| Fair regional pricing | PPP-adjusted prices for developing markets |
| Transparent | Total spend cap visible: "You've spent $X on EotS" |
