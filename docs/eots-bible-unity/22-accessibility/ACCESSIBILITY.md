# 22 — ACCESSIBILITY & INCLUSIVE DESIGN

## Visual Accessibility
- **Colorblind modes:** Deuteranopia, Protanopia, Tritanopia. Remaps all gameplay-critical colors. Enemy telegraphs use shape + color (never color alone).
- **High contrast mode:** Increases outline thickness on interactive elements, reduces background detail.
- **UI scaling:** 80% to 150% of default. All text remains readable at minimum scale on 720p.
- **Screen shake:** Adjustable intensity (0-100%), default 70%.
- **Flash reduction:** Option to reduce or eliminate screen flashes (infection tier transitions, crits).

## Motor Accessibility
- **Full input rebinding:** Every action remappable on keyboard, mouse, and controller independently.
- **One-handed mode:** All actions mappable to left-hand or right-hand only setups.
- **Hold vs toggle:** Every hold action (charge attack, Spore-Talk PTT) has a toggle alternative.
- **Auto-aim assist:** Adjustable aim magnetism for ranged classes (off/low/medium/high).
- **Difficulty assist:** "Symbiote Mode" — reduces enemy damage by 50%, increases I-frames by 50%, does not disable achievements but marks runs as assisted in leaderboards.

## Auditory Accessibility
- **Subtitles:** Full subtitle support for Mycelium Narrator, NPC dialogue, Spore-Talk transcription.
- **Visual sound indicators:** Directional indicators for off-screen audio cues (enemy alerts, Hunter proximity, Rite announcements).
- **Spore-Talk transcription:** Optional real-time speech-to-text display for VOIP (uses local device STT, not server-side).
- **Music/SFX/Voice:** Independent volume sliders for all buses.

## Cognitive Accessibility
- **Tutorial pacing:** Each mechanic introduced individually with dedicated room. No mechanic requires reading more than 2 sentences.
- **Minimap clarity:** Clear icons for room types. Current position always highlighted.
- **Boon descriptions:** Plain-language tooltips with exact numbers. No hidden mechanics.
- **Session length indicator:** Estimated remaining time displayed when entering a zone.

## Photosensitivity
- **Photosensitivity warning:** Displayed on first boot with option to enable all protections.
- **Reduced effects mode:** Disables rapid color changes, reduces particle density, dims bioluminescence intensity.
