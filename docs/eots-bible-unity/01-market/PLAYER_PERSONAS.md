## Persona 1: "The Optimizer" — Alex, 24

| Attribute | Detail |
|-----------|--------|
| **Archetype** | Min-maxer, theory-crafter |
| **Plays** | Hades, Path of Exile, Slay the Spire |
| **Session** | 2-4 hours, daily |
| **Motivation** | Build discovery, leaderboard ranking, Heat mastery |
| **Platform** | PC (Steam), mechanical keyboard, discord active |
| **Spending** | $30 base + Battle Pass every season |
| **Discovery** | YouTube build guides, Reddit tier lists |
| **Retention hook** | Echo Boon combinations (28 Duo Echoes), Heat system (50 levels), Codex (300+ entries) |
| **Churn risk** | Perceived balance issues, stale meta |
| **Mitigation** | Bi-weekly balance patches, seasonal boon rotations |

1. **Week 1:** Clears all 8 zones with Sporeling, discovers basic boon synergies
2. **Week 2-3:** Tests all 4 classes, finds Bloomcaster Artillery + Glacine Shatter combo
3. **Month 1:** Heat 20, starts posting builds on Reddit, joins Discord theory channel
4. **Month 2-3:** Heat 40+, competing for first Heat 50 clear on server, streaming attempts
5. **Month 6:** Has unlocked 280/300 Codex entries, advocates for new boon additions
