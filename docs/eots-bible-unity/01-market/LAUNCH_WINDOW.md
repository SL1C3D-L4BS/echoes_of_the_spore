# Launch Window Analysis

## Optimal Launch Window: Q1 2028 (January-March)

### Reasoning

| Factor | Q1 2028 Advantage |
|--------|-------------------|
| **Post-holiday spending** | Players have gift cards, new hardware |
| **Competitor gap** | Major AAA releases cluster Q4; Q1 is indie-friendly |
| **Award eligibility** | Full year for Game Awards 2028 consideration |
| **Steam seasonal sale** | Summer Sale 2028 = 5-month tail for first discount |
| **Content pipeline** | 4 Battle Pass seasons by end of Year 1 |

### Avoid
- **September-November:** AAA release season, attention saturation
- **December:** Holiday spending goes to known franchises
- **June:** E3/Summer Game Fest noise drowns indie announcements

## Launch Day Checklist

### Week -2 (Pre-Launch)
- [ ] Gold build submitted to all platforms
- [ ] Day-1 patch prepared based on final beta feedback
- [ ] Review codes sent to 100+ outlets
- [ ] Embargo date set for 2 days pre-launch
- [ ] Launch trailer finalized and uploaded (unlisted)
- [ ] Steam page updated with final assets
- [ ] Discord event scheduled for launch stream

### Day 0
- [ ] Steam page goes live at 10:00 AM PT
- [ ] Console stores update (coordinated with Sony/Microsoft/Nintendo)
- [ ] Launch trailer published across all channels
- [ ] 24-hour launch stream begins on Twitch
- [ ] Press embargo lifts (reviews go live)
- [ ] Community team online for bug reports
- [ ] Server scaling verified (10x expected CCU headroom)

### Week +1
- [ ] Day-1 patch deployed
- [ ] Hotfix pipeline active (daily if needed)
- [ ] Community sentiment monitoring (Steam reviews, Discord, Reddit)
- [ ] Sales data analysis: regional performance, platform split
- [ ] Influencer content tracking: views, wishlists, conversions

## Early Access vs Full Release Decision

**Decision: Full Release (NOT Early Access)**

| Factor | EA Argument | Full Release Argument | Winner |
|--------|------------|----------------------|--------|
| First impression | Can iterate | Only one chance | Full |
| Review scores | "It's EA" deflects criticism | Reviews count from day 1 | EA |
| Community Tree | Needs server pop to test | Needs server pop to work | Tie |
| Revenue timing | Earlier cash flow | Higher per-unit at launch | Full |
| Competitor positioning | StS2 used EA successfully | Hades surprised with 1.0 | Full |
| Team capacity | Can fix in public | Must be polished | Full |

**Verdict:** Ship 1.0 with 8 zones, 4 classes, full Community Tree, and online co-op. The Community Tree mechanic requires critical mass of simultaneous players — an EA launch with low population would make it feel broken.
